# Sugam Darshan

AI-Powered Smart Pilgrimage & Crowd Intelligence Platform

## Features

- Smart Darshan Planner
- Crowd Forecasting
- Slot Booking
- Live Crowd Intelligence
- Temple Navigation
- Weather & Travel Assistance
- Festival Calendar
- Spiritual Companion
- Emergency SOS
- Missing Person Assistance
- Notifications Center

## Roles

### Devotee
Planning, booking, navigation, emergency assistance.

### Authority
Crowd monitoring, risk analysis, incident management.

### Admin
System administration and authority management.
